To build Chamo, you will need:

  • OCaml >= 3.12 installed (not tested with prior versions, but it should be ok),
  • findlib,
  • Lablgtk2 >= 2.16.0 installed with findlib,
  • pcre-ocaml >= 6.2.5 (not tested with previous versions but it should be ok),
  • Config-file >= 1.1,
  • LablGtk-extras >= 1.2.

Get the sources from git repository: git clone https://github.com/zoggy/chamo.git

Split (un)install targets to (un)install-lib and (un)install-bin, for Opam. Revert indentation lexer to previous state (better indentation of function calls). Still not perfect, though. Upgrade to Lablgtk 2.16.0. First release after extraction from Cameleon. Now use findlib.

Just type 

./configure
make
make install
This will install chamo, chamo.byte and mk-chamo in the same directory as ocamlc. This will also install the chamo findlib package.
