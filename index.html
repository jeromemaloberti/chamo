What is Chamo ?

Chamo is a source code editor, even if it can be used to edit any text file. A system of "views" allows to edit some kinds of files in specific views.

It is developped in OCaml and OCaml is also used to configure it, like Elisp is used for emacs.

Want to know more ? Check the Documentation and the FAQ.

Get the code

Chamo is hosted on Github. See the Installation page for releases.
