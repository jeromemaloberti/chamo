watch_and_insert ic insert watches on input channel ic and uses the given insert when text is available and read from the channel.
Returns the watch id
on_end : can be used to specify a function called when the channel is closed.
val run_and_read_in_buffer : string -> (string -> unit) -> (int -> unit) -> unit
run_and_read_in_buffer command insert f runs a command and display its output with the insert function. The function f is called when the command exits, and takes in parameter the exit code (or signal number if the command was killed or stopped).
