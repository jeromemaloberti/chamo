Dependencies

To build Chamo, you will need:

  • OCaml >= 3.12 installed (not tested with prior versions, but it should be ok),
  • findlib,
  • Lablgtk2 >= 2.14.2 installed with findlib,
  • pcre-ocaml >= 6.2.5 (not tested with previous versions but it should be ok),
  • Config-file >= 1.1,
  • LablGtk-extras >= 1.1.
Get the sources

Get the sources from git repository: 

git clone https://github.com/zoggy/chamo.git

Releases
2.0 (2012-04-20) First release after extraction from Cameleon. Now use findlib.
Compilation

Just type 

./configure
make
make install
This will install chamo, chamo.byte and ml-chamo in the same directory as ocamlc. This will also install the chamo findlib package.
